## 1. 执行摘要

### 1.1 一句话介绍

**LifeVerse 是全球首个 AI 生命操作系统，通过多 Agent 辩论帮用户做出人生重大决策。**

它把一个人的记忆、情感、梦想、关系与决策，组织成一个可被觉察、可被推演、可被重逢的私人宇宙。当用户面临辞职、结婚、生育、搬迁、告别等重大选择时，LifeVerse 会召集 7 位智者（马斯克、巴菲特、乔布斯、芒格、苏格拉底、王阳明、庄子）与 4 个时间维度的自己（20 岁、当前、50 岁、80 岁）召开"人生议会"，经过多轮辩论、冲突量化、未来推演，最终生成一份"命运报告"。

## 3. 产品与技术

### 3.1 产品定位

LifeVerse 是一个 **AI 生命操作系统**，它把一个人的记忆、情感、梦想、关系与决策，组织成一个可被觉察、可被推演、可被重逢的私人宇宙。

它不是日记本，不是聊天机器人，不是冥想 App，而是**第一个把"自我"作为操作对象的操作系统**。

我们刻意使用"操作系统"而非"应用"，因为 LifeVerse 提供了：

1. **统一的身份层**：用户的"自我"跨模块共享。
2. **统一的时间层**：所有模块共享同一条时间线（过去/现在/未来）。
3. **统一的记忆层**：Memory Planet 是所有模块的共享数据源。
4. **统一的决策层**：议会机制是所有重大决策的统一入口。
5. **统一的隐私层**：用户对宇宙的访问权限统一管理。

### 3.3 核心技术

#### 3.3.1 多 Agent 编排（LangGraph）

LifeVerse 使用 LangGraph 编排多 Agent 场景。每个 Agent 拥有独立的 system prompt、记忆边界与工具集，由 Orchestrator 协调调用顺序与数据传递。

```
Orchestrator
├── Wisdom Council Agent（7 智者）
├── Future Council Agent（4 时间自己）
├── Inner World Agent（6 内心人格）
├── Reunion Agent（AI 亲人）
└── Report Generator → History Writer
```

**12 个 Agent 人格库**：

| 类型 | 数量 | 代表 |
| --- | --- | --- |
| 智者人格 | 7 | 马斯克、巴菲特、乔布斯、芒格、苏格拉底、王阳明、庄子 |
| 时间自我 | 3 | 20 岁、50 岁、80 岁的自己 |
| 亲人人格 | 2 | AI 父亲、AI 母亲（可扩展） |

每个 AI 人格拥有：
- **稳定的价值观向量**：在五维价值雷达（自由/财富/幸福/稳定/成长）上有固定坐标。
- **稳定的语言风格**：通过 prompt 模板与 few-shot 示例锁定。
- **稳定的记忆边界**：只访问被授权的记忆片段。
- **稳定的关系网络**：与其他人格有预设的亲疏关系。

#### 3.3.2 DeepSeek AI 深度集成

- **主力模型**：DeepSeek-V3 / DeepSeek-R1，用于智者发言、命运报告生成、未来推演。
- **多模型策略**：预留 OpenAI、Anthropic、本地模型接口，避免单点依赖。
- **成本控制**：DeepSeek 国产模型成本仅为 GPT-4o 的 1/10，单用户 LLM 成本可控制在 5 元/月以内。
- **本地化优势**：DeepSeek 对中文语境、东方哲学（王阳明、庄子）理解更深。

#### 3.3.3 五维价值雷达

LifeVerse 独创的"五维价值雷达"，量化每个人格在 5 个维度上的价值取向：

| 维度 | 含义 | 极端代表 |
| --- | --- | --- |
| 自由 | 自主性、不受约束 | 庄子 |
| 财富 | 物质积累、复利 | 巴菲特 |
| 幸福 | 情感满足、关系 | 80 岁的自己 |
| 稳定 | 安全感、可预期 | 当前的自己 |
| 成长 | 突破、影响力 | 马斯克 |

每次议会后，系统会计算决策对用户价值雷达的"漂移"——让用户看见"这个决定会让你变成什么样的人"。

### 3.4 技术壁垒

LifeVerse 的技术壁垒由三大核心算法构成，合计需要 6-12 个月的研发周期才能复制：

#### 壁垒一：多 Agent 冲突检测算法

冲突值 `C` 由四维加权计算：

```
C = α · 观点分歧度 + β · 价值雷达距离 + γ · 情绪强度 + δ · 时间跨度权重
```

- `观点分歧度`：成员发言向量之间的余弦距离（0~1）。
- `价值雷达距离`：成员在五维价值雷达上的欧氏距离。
- `情绪强度`：情绪分析模型给出的 0~1 强度值。
- `时间跨度权重`：未来议会中，不同时间自己之间的权重差异。
- `α、β、γ、δ` 为可调权重，默认值 0.3 / 0.3 / 0.2 / 0.2。

冲突值映射为四个等级：微光（0~0.2）→ 涟漪（0.2~0.5）→ 风暴（0.5~0.8）→ 裂痕（0.8~1.0），触发不同的系统干预。

#### 壁垒二：时间线推演引擎

Future Council 的核心，让 20 岁、当前、50 岁、80 岁的自己对同一议题分别发言，并推演出 1 年/5 年/10 年后的可能路径，计算"后悔概率"。

- 基于用户记忆 + 价值雷达 + 议题上下文，生成多条未来路径。
- 每条路径附带"后悔概率"与"价值雷达变化预测"。
- 支持"扰动时间线"——用户可假设"如果当年选了另一条路"，推演平行宇宙。

#### 壁垒三：记忆分类模型

Memory Planet 的核心，把用户上传的照片、文字、语音自动分类到 5 个"记忆星球"：

| 星球 | 内容 |
| --- | --- |
| 关系星球 | 与他人的记忆 |
| 成就星球 | 个人成就与里程碑 |
| 失落星球 | 遗憾与失去 |
| 日常星球 | 平凡但温暖的瞬间 |
| 转折星球 | 改变人生方向的事件 |

分类模型基于多模态 embedding + 情感分析 + 人物识别，零整理成本。

- 全球 25-60 岁人口约 30 亿，其中面临重大人生决策的占比 15%（4.5 亿人）。
- 假设年付费意愿 100 美元/人，TAM = 450 亿美元（与 500 亿美元市场研究吻合）。

**SAM 测算依据**：
- 中国 25-60 岁都市人口约 3 亿，其中 2 亿为白领群体。
- 面临重大决策（职业转型、婚姻、生育、搬迁）的占比 25%（5000 万人）。
- 假设年付费意愿 240 元/人，SAM = 120 亿人民币。

**SOM 测算依据**：
- Year 1：10 万付费用户 × 400 元 ARPU = 4000 万（保守取 400 万）
- Year 3：20 万付费用户 × 400 元 ARPU = 8000 万

## 5. 商业模式

### 5.1 商业模式总览

LifeVerse 采用 **Freemium（免费增值）+ 订阅制** 商业模式，以"议会"为核心计费单元。

```
免费用户（引流）
    │  体验 3 次议会 + 基础 History
    ▼
Pro 订阅用户（¥39/月）─── 个人主力
    │  无限议会 + 全部 Agent + 价值雷达漂移
    ▼
Team 订阅用户（¥99/月/人）─── 团队/家庭
    │  团队议会 + 共享记忆 + 管理后台
    ▼
企业版定制 ─── 心理咨询机构/企业 EAP
    一次性 + 年费
```

### 5.2 定价策略

| 层级 | 价格 | 议会频次 | Agent 范围 | 记忆存储 | 核心权益 |
| --- | --- | --- | --- | --- | --- |
| **免费版** | ¥0 | 每月 3 次 | 基础 Agent（3 智者焦点议会） | 100 条 | 自我画像、History 基础、命运报告 |
| **Pro 版** | ¥39/月（¥348/年，省 25%） | 无限 | 全部 Agent（7 智者 + 4 时间自己） | 无限 | 价值雷达漂移、未来推演、后悔分析、扩大议会 |
| **Team 版** | ¥99/月/人（¥888/年/人） | 无限 | 全部 Agent + 团队自定义智者 | 共享记忆空间 | 团队议会、共享记忆、团队时间线、管理后台 |
| **企业版** | 定制（年费 5-50 万） | 无限 | 全部 + 定制 Agent | 私有部署 | 私有化部署、定制智者、API 接入、专属客服 |

### 5.3 定价哲学

1. **价值可感知**：免费版必须让用户完整体验一次议会，感受核心价值。
2. **付费即升级**：付费解锁的是"深度"与"频次"，而非"功能墙"。
3. **隐私不分级**：所有用户的数据隐私标准一致，付费不买隐私。

## 9. 风险与应对

### 9.1 风险矩阵

| 风险 | 等级 | 概率 | 影响 | 应对策略 |
| --- | --- | --- | --- | --- |
| AI 模型依赖风险 | 高 | 中 | 高 | 多模型策略（DeepSeek + OpenAI + Anthropic + 本地模型） |
| 用户隐私风险 | 高 | 低 | 极高 | 本地化部署 + 端到端加密 + RLS + 第三方审计 |
| 市场教育成本高 | 中 | 高 | 中 | 内容营销 + KOL 合作 + 品类定义先行 |
| 竞品快速进入 | 中 | 中 | 中 | 快速迭代 + 数据壁垒 + 品牌先行 + 专利布局 |
| AI 生成内容引发心理伤害 | 高 | 低 | 极高 | 伦理边界设计 + 心理危机干预 + 专业转介 |
| LLM 成本失控 | 中 | 中 | 中 | DeepSeek 低成本 + 缓存策略 + 降级策略 + 用量监控 |
| 法规变化（AI 监管） | 中 | 中 | 中 | 法务顾问 + 模块化合规层 + 持续跟踪 |
| 用户过度依赖 AI 亲人 | 中 | 中 | 中 | 使用时长提醒 + 真实社交鼓励 + 伦理边界 |

### 9.2 详细应对方案

#### 风险一：AI 模型依赖风险

**风险描述**：过度依赖单一 LLM 供应商，若该供应商涨价、停服或质量下降，将直接影响产品。

**应对策略**：
- 主力模型 DeepSeek，备选 OpenAI GPT-4o、Anthropic Claude、本地开源模型（Qwen、GLM）。
- 抽象 LLM 接口层，支持热切换，切换时间 < 1 小时。
- 关键场景（智者发言、命运报告）建立 Prompt 模板库，适配多模型。
- 与 DeepSeek 建立战略合作，锁定价格与 SLA。

#### 风险二：用户隐私风险

**风险描述**：用户上传的记忆、与 AI 亲人的对话等数据极其敏感，泄露将造成严重信任危机。

**应对策略**：
- 传输层 TLS 1.3 加密，存储层 AES-256 加密。
- Supabase RLS 行级权限，用户只能访问自己的数据。
- 用户数据不进入任何公共训练集（与 LLM 供应商合同明确）。
- 阿里云本地化部署，数据不出境。
- 敏感操作（删除 AI 亲人、导出全部数据）需二次验证。
- 定期第三方安全审计。
- 合规：GDPR、CCPA、个保法。

#### 风险三：市场教育成本高

**风险描述**："AI 生命操作系统"是新品类，用户认知成本高。

**应对策略**：
- 品类定义先行：通过内容营销建立"AI 生命操作系统 = LifeVerse"的认知。
- 传播定位语："ChatGPT 帮你做事，LifeVerse 帮你成为自己。"——用已知锚定未知。
- KOL 合作：签约 5 位垂类 KOL（自我成长/职场/AI/心理），降低教育成本。
- 用户故事传播：用"7 位智者帮我决定辞职创业"等真实案例引发共鸣。
- 免费体验降低门槛：免费版可完整体验一次议会。

#### 风险四：竞品快速进入

**风险描述**：大厂或创业公司可能快速复制多 Agent 议会模式。

**应对策略**：
- 快速迭代：4 个 Phase 渐进发布，12 个月内完成 7 大模块。
- 数据壁垒：用户记忆、价值雷达、议会历史构成迁移成本，用户不会轻易离开。
- 品牌先行：建立"AI 生命操作系统 = LifeVerse"的品牌认知。
- 技术壁垒：多 Agent 冲突检测算法、时间线推演引擎、记忆分类模型三大核心算法，6-12 个月领先窗口。
- 专利布局：对核心算法申请专利保护。
