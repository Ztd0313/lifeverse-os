## 1. 投资人画像总览

### 1.1 适合 LifeVerse 的三类投资人

LifeVerse 处于 AI、心理健康、消费互联网三个赛道的交叉点，因此适合三类投资人：

| 类型 | 关注点 | 匹配度 | 优先级 |
| --- | --- | --- | --- |
| AI 赛道投资人 | 技术壁垒、模型能力、Agent 架构 | 高 | P0 |
| 心理健康赛道投资人 | 伦理边界、用户安全、社会价值 | 高 | P0 |
| 消费互联网投资人 | 用户增长、留存、商业化 | 中高 | P1 |

### 1.2 理想投资人特征

LifeVerse 的理想投资人应具备以下特征：

| 维度 | 理想特征 |
| --- | --- |
| 阶段偏好 | 天使轮/Pre-A，愿意陪跑 3-5 年 |
| 赛道认知 | 理解 AI 应用层 + 心理健康 + 消费互联网 |
| 价值观认同 | 认同"用 AI 帮助人理解自己"的社会价值 |
| 资源加持 | 能提供 AI 技术、心理学专家、媒体资源 |
| 决策风格 | 数据驱动，但愿意为品类开创者下注 |
| 退出预期 | 接受 5-7 年退出周期，不追求短期套现 |

### 1.3 不适合的投资人

| 类型 | 不适合原因 |
| --- | --- |
| 纯财务投资人（无产业资源） | 无法提供战略价值，且可能施压短期变现 |
| 追求 2 年内退出的投资人 | LifeVerse 需要 3-5 年成长周期 |
| 不理解 AI 的传统投资人 | 无法理解技术壁垒与产品价值 |
| 对"AI 心理"有伦理顾虑的投资人 | 价值观不匹配，后续合作摩擦大 |
| 要求控股权的投资人 | 天使轮出让不超过 10%，不接受控股权要求 |

## 2. AI 赛道投资人

### 2.1 红杉资本中国（Sequoia Capital China）

| 维度 | 详情 |
| --- | --- |
| 机构性质 | 顶级 VC |
| 管理规模 | 数百亿美元 |
| AI 投资偏好 | 基础模型、AI 应用层、AI Infra |
| 典型 AI 案例 | 字节跳动、大疆、Shein、Moonshot（月之暗面） |
| 决策风格 | 顶级赛道 + 顶级团队，决策快，但标准极高 |
| 单笔投资 | 天使轮 300-1000 万人民币 |
| 关注点 | 赛道天花板、团队背景、技术壁垒、数据飞轮 |

**对 LifeVerse 的关注点**：
- 多 Agent 架构的技术深度（LangGraph 编排、冲突检测算法）
- 是否能成为"AI 自我探索"赛道的品类定义者
- 数据飞轮：用户记忆 → 议会质量提升 → 更多用户
- 与 DeepSeek 的战略合作关系

**对接策略**：
- 切入点：通过红杉被投企业（如 Moonshot）引荐。
- 话术重点：强调"AI 生命操作系统"是 AI 应用层的下一个品类机会，类比"ChatGPT 是通用 AI，LifeVerse 是垂直 AI"。
- 亮点：MVP 已完成 + 12 个 Agent 人格库 + 三大核心算法。
- 注意：红杉标准极高，需准备极高质量的材料，建议作为"标杆投资人"优先对接，即便不投也能背书。

### 5.4 融资策略原则

1. **制造竞争性**：同时对接 3-5 家投资人，制造竞争氛围，提升估值谈判筹码。
2. **先快后慢**：先锁定决策快的（真格、奇绩），再对接决策慢的（红杉、高瓴）。
3. **背书优先**：即便某些大机构天使轮不投，也要建立联系，为 A 轮铺垫。
4. **金额灵活**：500 万可以 1 家领投，也可以 2-3 家合投（如真格 300 万 + 险峰 200 万）。
5. **不过度稀释**：天使轮出让不超过 10%，为 A 轮预留空间。
