## 1. 品牌总览

### 1.1 品牌名称

- 英文：**LifeVerse**
- 中文：**生命宇宙**（品牌名以英文 LifeVerse 为主，中文场景可辅以"生命宇宙"释义）
- 含义：Life（生命）+ Verse（宇宙/诗篇），寓意"每一个生命都是一个宇宙，也是一首诗"。

### 1.2 品牌口号

- 英文：**Every life deserves its own universe.**
- 中文：**每一个生命，都值得拥有自己的宇宙。**

### 1.3 品牌定位

AI 生命操作系统——帮你在重大决策时与智者、记忆和未来的自己对话。

### 1.4 品牌个性

| 个性维度 | 描述 |
| --- | --- |
| 智慧 | 不是聪明，而是深邃。像苏格拉底一样追问，像庄子一样通透。 |
| 温暖 | 不是热情，而是温柔。像一面温暖的镜子，照见你但不评判你。 |
| 神秘 | 不是晦涩，而是深邃。像宇宙一样，让人想探索。 |
| 真实 | 不虚构答案，呈现多元视角。镜子而非裁判。 |

### 1.5 品牌隐喻

**宇宙**。LifeVerse 的所有视觉与语言都围绕"宇宙"隐喻展开：
- 用户的生命 = 一个宇宙
- 记忆 = 星球
- 议会 = 星辰汇聚
- 时间线 = 星河
- 价值雷达 = 星图坐标

## 2. Logo 规范

### 2.1 Logo 概念

LifeVerse 的 Logo 以"生命宇宙"为核心概念：一个由 7 颗星辰环绕的中心光点，象征 7 位智者围绕用户的"自我"召开议会，整体构成一个微型宇宙。

### 2.2 Logo 构成

| 元素 | 说明 |
| --- | --- |
| 中心光点 | 代表用户的"自我"，是宇宙的中心 |
| 7 颗星辰 | 代表 7 位智者，环绕中心，各有轨道 |
| 轨道环 | 代表时间线，星辰沿轨道运行 |
| 整体形态 | 圆形，象征完整与宇宙 |

### 2.3 Logo 版本

| 版本 | 用途 | 说明 |
| --- | --- | --- |
| 主 Logo（彩色） | 官网、正式场合 | 深空蓝底 + 金色星辰 + 白色光点 |
| Logo（白色） | 深色背景 | 白色单色版本 |
| Logo（黑色） | 浅色背景 | 深空蓝单色版本 |
| Logo Mark（图标） | Favicon、App 图标 | 仅中心光点 + 7 星辰，无文字 |
| Logo 横版 | 页眉、文档 | 图标 + "LifeVerse" 文字横排 |
| Logo 竖版 | 社交媒体封面 | 图标 + "LifeVerse" 文字竖排 |

### 2.4 Logo 使用规范

#### 2.4.1 最小尺寸

| 应用场景 | 最小尺寸 |
| --- | --- |
| 印刷 | Logo 宽度 ≥ 25mm |
| 屏幕（Logo Mark） | 24 × 24 px |
| 屏幕（完整 Logo） | 120 × 40 px |
| Favicon | 16 × 16 px（仅 Logo Mark） |

#### 2.4.2 安全区域

- Logo 四周需保留不小于 Logo 高度 1/2 的空白区域。
- 安全区域内不可放置任何文字、图形或其他元素。

#### 2.4.3 禁止用法

| 禁止 | 说明 |
| --- | --- |
| 不可拉伸变形 | 保持原始比例 |
| 不可旋转 | Logo 始终保持正向 |
| 不可更改颜色 | 仅可使用规范中的颜色版本 |
| 不可添加阴影/描边 | 保持简洁 |
| 不可放置在复杂背景上 | 确保对比度与可读性 |
| 不可拆分使用 | 图标与文字不可分离使用（Logo Mark 除外） |

#### 2.4.4 背景适配

| 背景类型 | 推荐版本 |
| --- | --- |
| 白色/浅色背景 | 主 Logo（彩色）或 Logo（黑色） |
| 深色/黑色背景 | Logo（白色） |
| 品牌色背景 | Logo（白色） |
| 图片背景 | 需加半透明遮罩后使用 Logo（白色） |

## 4. 字体规范

### 4.1 字体家族

| 用途 | 中文字体 | 英文字体 | 备选 |
| --- | --- | --- | --- |
| 标题 | 思源黑体 Bold | Inter Bold | PingFang SC / SF Pro |
| 正文 | 思源黑体 Regular | Inter Regular | PingFang SC / SF Pro |
| 数字 | - | Inter Medium | SF Pro |
| 特殊（品牌/引言） | 思源宋体 Light | Playfair Display Light | - |

### 4.2 字号规范

| 层级 | 屏幕（Web） | 屏幕（移动端） | 印刷 |
| --- | --- | --- | --- |
| H1 主标题 | 48px / 3rem | 32px | 36pt |
| H2 章节标题 | 36px / 2.25rem | 28px | 28pt |
| H3 小标题 | 24px / 1.5rem | 20px | 20pt |
| H4 段落标题 | 20px / 1.25rem | 18px | 16pt |
| Body 正文 | 16px / 1rem | 15px | 12pt |
| Caption 注释 | 14px / 0.875rem | 13px | 10pt |
| Tiny 微小 | 12px / 0.75rem | 11px | 8pt |

### 4.3 字重规范

| 字重 | 用途 |
| --- | --- |
| Light（300） | 特殊引言、品牌大字 |
| Regular（400） | 正文 |
| Medium（500） | 次要标题、数字 |
| Bold（700） | 主标题、CTA 文字 |

### 4.4 行高与字间距

| 用途 | 行高 | 字间距 |
| --- | --- | --- |
| 标题 | 1.2 | -0.02em |
| 正文 | 1.6 | 0 |
| 注释 | 1.5 | 0 |

### 4.5 字体使用原则

- 中英文混排时，中文用思源黑体，英文与数字用 Inter，确保视觉统一。
- 品牌大字与引言可使用思源宋体/Playfair Display，增加诗意感。
- 数字（如冲突值、雷达坐标）统一使用 Inter Medium，确保清晰。
- 禁止使用超过 2 种字体家族，保持简洁。

## 7. 宣传视频脚本

### 7.1 30 秒预告片脚本

**视频名称**：Every life deserves its own universe
**时长**：30 秒
**用途**：全渠道预热投放、Product Hunt 发布、社交媒体传播
**配乐**：空灵钢琴 + 弦乐渐强
**色调**：深空蓝 + 暖金光点
**画幅**：16:9（主版）+ 9:16（竖版）+ 1:1（方版）

```
[0:00-0:03] 黑屏渐亮
画面：纯黑背景，白色文字逐字浮现
文字："每一个生命，"
文字："都值得拥有自己的宇宙。"
音效：单音钢琴，空灵

[0:03-0:08] 宇宙展开
画面：从中心光点向外扩散，宇宙星图缓缓展开
镜头缓慢推进，7 颗星辰依次亮起
每颗星辰旁浮现智者名字（淡入淡出）：
"马斯克" "巴菲特" "乔布斯" "芒格" "苏格拉底" "王阳明" "庄子"
音效：钢琴渐强，加入弦乐

[0:08-0:14] 议会意象
画面：黑暗中一个人形剪影（极光白），面前浮现 7 个光点（星辰金）
光点依次闪烁，模拟发言（无声音，字幕闪现）：
"你为什么想创业？"
"你的能力圈在哪？"
"无用之用，方为大用。"
音效：弦乐持续，加入低频脉动

[0:14-0:19] 时间维度
画面：时间轴从左到右展开，4 个人形剪影出现
标注："20岁" "现在" "50岁" "80岁"
80 岁的剪影缓缓转身，面向镜头
字幕（手写体）："别怕，这件事没那么重要。"
音效：弦乐达到小高潮后回落

[0:19-0:24] 记忆星球
画面：一颗星球旋转，表面照片如光点般亮起
星球汇聚，形成一张宇宙地图
字幕："你的记忆，就是你的宇宙。"
音效：钢琴主旋律回归，温暖

[0:24-0:28] Logo 定格
画面：所有元素汇聚为中心光点，LifeVerse Logo 浮现
Logo 下方文字："AI 生命操作系统"
音效：最后一个和弦，余音渐弱

[0:28-0:30] 结尾
画面：Logo 下方浮现
文字："Every life deserves its own universe."
文字："即将上线。"
音效：静默
```

### 7.2 3 分钟演示视频脚本

**视频名称**：LifeVerse · 3 分钟产品演示
**时长**：3 分钟
**用途**：官网首页、Product Hunt、投资人演示、媒体通稿
**配乐**：前半段紧张（决策困境），后半段释然温暖（获得洞察）
**画幅**：16:9
**旁白**：专业温暖男声

```
[0:00-0:15] 开场 · 困境
画面：城市夜景，高楼一盏灯。镜头推进，一个年轻人（林然）对着电脑发呆，桌上是辞职信草稿。
旁白："凌晨 2 点，28 岁的林然在纠结一件事——要不要辞职创业。"
旁白："朋友说'稳住'，家人说'别折腾'，他自己……说不清。"
字幕：LifeVerse · AI 生命操作系统

[0:15-0:20] 引入
画面：林然打开 LifeVerse，深空蓝界面展开，Logo 浮现。
旁白："他决定，换一种方式做这个决定。"

[0:20-0:55] 场景一 · 智慧议会
画面：林然输入议题"我是否应该辞职创业？"
画面：7 位智者头像依次亮起（深空蓝背景 + 星辰金光点），发言以流式文字呈现。
旁白："他让 7 位智者，为他召开一场议会。"

[智者发言，配字幕]
马斯克："用第一性原理看，你的核心能力在产品，不在大厂流程。"
```

## 9. 品牌素材管理

### 9.1 素材存储

- 所有品牌素材存储在团队云盘的 `Brand Assets` 文件夹中。
- 按类型分子文件夹：Logo / Color / Typography / Social / Video / Screenshot / Template。
- 每个文件命名规范：`类型_版本_尺寸_日期`，如 `Logo_Primary_1920x1080_20261201.png`。

### 9.2 版本管理

- 品牌素材每次更新需保留旧版本，存入 `Archive` 文件夹。
- 重大品牌升级（如 Logo 改版）需发布版本说明，通知全团队。
- 所有对外使用的品牌素材必须从最新版本文件夹中取用。

### 9.3 使用授权

| 素材类型 | 内部使用 | 外部合作方 | 媒体 |
| --- | --- | --- | --- |
| Logo | 自由使用 | 需签署品牌使用协议 | 提供 Media Kit |
| 品牌色/字体 | 自由使用 | 需指导使用 | 提供 Media Kit |
| 产品截图 | 自由使用 | 需审核 | 提供 Media Kit |
| 宣传视频 | 自由使用 | 需授权 | 可引用 |
| PPT 模板 | 自由使用 | 不对外 | 不对外 |
