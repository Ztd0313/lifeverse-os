## 1. 职责

品牌叙事 Skill 负责 LifeVerse OS 的品牌故事构建与传播内容创作，包括品牌故事、宣传文案、视频脚本、社交媒体内容与产品叙事。该 Skill 将 LifeVerse 的产品定位转化为打动人心的故事，建立用户与品牌的情感连接。

### 核心职责
- 撰写品牌故事
- 创作宣传文案
- 编写视频脚本
- 撰写社交媒体内容
- 产品叙事设计
- 品牌语调定义
- 传播策略建议

## 4. LifeVerse 品牌核心

### 4.1 品牌定位
LifeVerse 是 AI 生命操作系统——一个由 7 位 AI 智者组成的议会，为人生的每一个重大决策提供多元视角的深度思辨。

### 4.2 品牌主张
「你的人生，值得一场议会。」

### 4.3 品牌价值观
- 多元：每个声音都值得被听见
- 深度：超越表面，触及本质
- 成长：每一次思辨都是一次成长
- 温暖：科技有温度，决策不孤独

### 4.4 品牌语调
- 智慧但不居高临下
- 深度但不晦涩难懂
- 温暖但不矫揉造作
- 专业但不冷冰冰

## 7. 工作流程

### 阶段 1：品牌理解
1. 深入理解 LifeVerse 产品定位
2. 理解目标用户画像与情感需求
3. 分析竞品传播策略
4. 明确传播目标
5. 输出：品牌理解摘要

### 阶段 2：品牌语调定义
1. 定义品牌人格特质
2. 确定语言风格（正式/亲切/激励）
3. 制定用词偏好与禁忌
4. 创建语调示例（正确 vs 错误）
5. 输出：品牌语调指南

### 阶段 3：品牌故事创作
1. 确定故事主题与情感弧线
2. 构建「困境-灵感-构建-体验-愿景」叙事结构
3. 撰写故事初稿
4. 打磨语言与节奏
5. 输出：品牌故事文档

### 阶段 4：宣传文案创作
1. 根据渠道特性撰写文案
2. 官网文案：Hero 标题 + 副标题 + 功能介绍 + CTA
3. 社媒文案：短平快，引发共鸣
4. 邮件文案：个性化，温暖引导
5. 输出：多渠道文案集

### 阶段 5：视频脚本创作
1. 确定视频目标与时长
2. 设计分镜结构
3. 撰写画面描述与旁白
4. 标注字幕与音效
5. 输出：视频分镜脚本

### 阶段 6：社媒内容规划
1. 制定内容主题日历
2. 按主题创作系列内容
3. 设计互动话题
4. 规划发布节奏
5. 输出：社媒内容日历

### 阶段 7：交付与反馈
1. 交付所有品牌素材
2. 收集用户反馈
3. 迭代优化文案
4. 输出：最终版素材

## 10. 质量标准

- 品牌故事情感真挚，引发共鸣
- 文案符合品牌语调指南
- 视频脚本分镜清晰，可执行
- 社媒内容有传播性，引发互动
- 所有内容无错别字，语言流畅
- 内容覆盖品牌认知到用户留存全链路
- 文案有明确的 CTA（行动号召）
